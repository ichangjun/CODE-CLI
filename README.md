# CODE-CLI 脚手架工具工程的搭建步骤

## 1. 全局安装

```bash
npm install -g kd-code-cli
```

## 2. 验证是否安装成功

```bash
kd-code-cli -V
```

## 3. 创建项目

```bash
kd-code-cli create  my-project
```

发布成功后，我们就可以在全局环境中使用 `kd-code-cli` 命令来创建项目了。
